Description

Enter NCM Description in a should be is format. For NCMs not related to product discrepancies (Eg: Calibration), enter enough detail so one can ascertain the intention.

Make sure to enter the correct NCM Type: Calibration for Calibration, Return for Returns et al.

Subject Codes

  • HFS - Held For Statement

    Use this for paperwork issues.

  • DAM - Damaged
  • FRM - Form
  • FIT - Fit
  • FUN - Function
  • WKM - Workmanship

    Use this for workmanship issues that do not fit into form/fit/function subjects (Eg: Burrs).

Cause Codes

If you don't know, just enter TBD

  • "HE" - Human Error.
  • "ME" - Machine Error.
  • "MA" - Material Error.
  • "DE" - Design Error.
  • "VE" - Vendor Error.
  • "OT" - Other.
  • "PC" - Process Control (Did not follow process)
  • "UD" - Undetermined

Disposition

Enter the Disposition of the NCM. This is the action that will be taken to resolve the NCM. Eg: Rework, Scrap, Return to Vendor, etc.

Verification

Enter the Verification of the NCM. This is the action that has been taken to verify the NCM has been resolved.

Enter a tracking number as applicable for returns

Enter a WO number for new rework orders.
